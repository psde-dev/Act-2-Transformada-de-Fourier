# Simulación y análisis de señales con la transformada de Fourier

Este proyecto analizará señales con respectp al tiempo y la frecuencia mediante el uso de la Transformada de Fourier. Se generan señales básicas como una señal senoidal, un pulso rectangular y una función escalón, las cuales son representadas gráficamente en ambos dominios.

## Requisitos

- Python3
- NumPy
- Matplotlib

## Contenido

- Generación de señales con respecto al dominio del tiempo.
- Cálculo de la Transformada de Fourier mediande la función FFT.
- Verificar la propiedad de linealidad.
- Visualización del especto de la frecuencia de cada señal.

## Ejecutar

Para ejecutar el archivo se debe hacerlo con:

python3 TdF.py
